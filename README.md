# Portfolio Contact API

A simple REST API built using Node.js and Express.js for handling contact form submissions.

## Features
- REST API
- JSON request handling
- Contact form endpoint
- Error handling

## Technologies Used
- Node.js
- Express.js

## API Endpoints

### GET /
Returns a welcome message.

### POST /contact
Accepts contact form data and returns a success response.
